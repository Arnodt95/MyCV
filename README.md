# Arno du Toit | Online CV

A clean, responsive **single-page online CV** built with **HTML + Bootstrap 5** (via CDN) and a small amount of custom CSS.

## Preview
Open `index.html` in any modern browser.

## Features
- Responsive layout (Bootstrap grid + utility classes)
- Sticky sidebar on larger screens
- Print-friendly styles (`@media print`)
- “Skip to content” accessibility link
- Auto-updating footer year (small inline JavaScript)

## Files
- `index.html` — main CV page
- `profile.png` — profile image (replace with your own)

## Customize
Update these placeholders in `index.html`:
- Phone link: `tel:+27XXXXXXXXX`
- Email link: `YOUR_EMAIL@example.com`
- Content sections: About, Experience, Education, Projects, Skills, etc.

## Dependencies
- Bootstrap 5 loaded from CDN:
  - CSS: `https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css`
  - JS: `https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js`
